🛍️ Simple E-Commerce System in Python (OOP)
This is a basic Object-Oriented Programming (OOP) based E-Commerce simulation using Python. It allows you to create products, customers, manage carts, and place orders.

💡 Features
Product creation with price and stock
Customer creation with shopping cart
Add products to cart
Calculate total cost
Place order and update stock
Order summary display
🧱 Technologies Used
Python 3
Object-Oriented Programming (OOP)
🚀 How to Run
Clone the repository:
git clone https://github.com/yourusername/python-ecommerce-oop.git
cd python-ecommerce-oop
